## What is Inventory 
typically ``Inventory`` or is a file which used By Control Node to understand and manage the manage node in ansible. you can write ``Inventory`` in ``Yaml`` or ``.ini``

You can put this ``.ini`` anywhere but it's recommended to place in folder ``etc/Ansible/host`` 

It contain a list of IP address or Host name (manage nodes) you want to control.
